# CLI Personal Expense Tracker

A command-line application built in Python to track, manage, and analyze personal expenses using Object-Oriented Programming principles.

## Features
- Add, view, update, and delete expense records
- Persistent storage using JSON file handling
- Exception handling for invalid inputs
- Clean OOP structure with dedicated `Expense` and `ExpenseTracker` classes

## Project Structure

- `main.py` — Entry point, handles CLI menu and user interaction
- `tracker.py` — ExpenseTracker class, manages the collection of expenses
- `expense.py` — Expense class, represents a single expense record

## Tech Stack
- Python 3
- JSON for data persistence
- Object-Oriented Programming (OOP)

## How to Run

    python main.py

## What This Project Demonstrates
- Class design and OOP principles (encapsulation, separation of concerns)
- File I/O and JSON serialization
- CRUD operations (Create, Read, Update, Delete)
- Exception handling for robust input validation
